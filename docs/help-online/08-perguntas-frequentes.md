# Perguntas frequentes (FAQ)

## Geral

**Preciso de instalar algo no PC?**  
Não. A aplicação corre no browser; só precisa de rede até ao servidor da instituição.

**Posso ter vários casos abertos para o mesmo NIF?**  
A plataforma pode mostrar histórico por NIF na lista; confirme a política interna para reutilização vs. novo caso.

**O que é o score 0–100?**  
Síntese automática de risco com base nos sinais. Não substitui a decisão humana (RGPD Art. 22).

## Triagem

**Quanto demora a triagem?**  
Tipicamente 2–10 minutos conforme número de partes e carga do servidor. Casos complexos com muitos UBOs podem demorar mais.

**Posso fechar o browser durante a triagem?**  
Sim. A triagem corre no servidor. Reabra o detalhe do caso para ver o resultado.

**Refazer triagem apaga as minhas confirmações de sinais?**  
A re-triagem regenera sinais automáticos; confirmações manuais anteriores podem ser afectadas conforme regras do motor — reveja sempre após re-triagem.

## Conformidade

**Todos os UBOs têm de estar verificados?**  
Sim, para aprovar, quando existem partes com papel UBO ou órgão social no caso.

**Quando devo comunicar SAR?**  
Quando identificar operação suspeita nos termos da Lei 83/2017. O banner amarelo sugere; a decisão é do analista/compliance.

**O que é EDD?**  
Due diligence reforçada — exige origem dos fundos e segundo aprovador.

## Relatórios

**O relatório serve para arquivo regulatório?**  
É um relatório operacional KYC com fundamentação automática. Valide com compliance se cumpre todos os requisitos internos.

**Posso editar o relatório no sistema?**  
O relatório é gerado automaticamente; use notas no caso, sinais manuais e audit trail para complementos.

## Perfis

**Sou analista mas não vejo Administração**  
Normal — menu Admin só para `KYC.Admin` (e auditoria para `KYC.Auditor`).

**Como peço acesso?**  
Contacte o administrador da aplicação na sua instituição (role Entra ID).
