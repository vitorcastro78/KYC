# Funcionalidades por área de negócio

Resumo das capacidades da plataforma para o dia-a-day do analista. Para o inventário completo com IDs e base legal, consulte a documentação de homologação da instituição.

## Gestão de casos

| Funcionalidade | Onde na app |
|----------------|-------------|
| Abrir caso com validação PAC | Novo caso |
| Listar e filtrar casos | Casos KYC |
| Aprovar / rejeitar / revisão | Detalhe do caso |
| Atribuição e revisão periódica | Automático após aprovação |

## Entidades e UBO

| Funcionalidade | Onde na app |
|----------------|-------------|
| Resolução GLEIF / dados societários | Detalhe do caso (cartão empresa) |
| Grafo UBO interactivo | Detalhe → Grafo UBO |
| Adicionar parte manual | Detalhe → Adicionar parte |
| Reportar discrepância RCBE | Painel de identidade da parte |

## Triagem e risco

| Funcionalidade | Onde na app |
|----------------|-------------|
| Triagem automática completa | Ao criar / Refazer triagem |
| Progresso em tempo real | Barra no detalhe |
| Triagem por parte | Cartão da parte |
| Confirmar / descartar sinais | Lista de sinais |
| Sinal manual | Registar sinal manual |
| Scoring 0–100 e nível | Hero do caso |
| DDC (simplificada / standard / EDD) | Secção conformidade |

## Legenda de estados (integrações)

- **Funcional** — disponível na sua instalação.
- **Contingência manual** — use os campos de registo manual quando a API externa falhar.
- **Administrador** — requer perfil Admin ou apoio de TI.
